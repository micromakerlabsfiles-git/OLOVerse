<p align="center">
The Official Development Hub for the OLOVerse Ecosystem

OLOVerse is an ecosystem of interconnected smart devices, desktop companions, embedded software, firmware, and web applications developed by **Micromaker Labs**.

This organization hosts the official firmware, web controllers, utilities, documentation and development tools for every OLOVerse product.

# Repositories

## 🟠 OLO-Bit

The flagship smart desktop companion firmware. Features 50+ animated OLO faces, retro games (Snake, Dino, Flappy), phone notification sync, BLE weather and maps, alarms, timers, LED control, and a browser-based web controller.

Repository

https://github.com/micromakerlabsfiles-git/OLO-Bit

Web Controller

https://micromakerlabsfiles-git.github.io/OLO-Bit/

---

## ✈️ OLO-Aero

Aviation radar desk companion. Tracks real flights in real-time using the OpenSky API. Shows callsign, altitude, speed, heading, distance, and origin airport. Falls back to an offline radar simulator and animated world clock when no flights are detected.

Repository

https://github.com/micromakerlabsfiles-git/OLO-Aero

Web App

https://micromakerlabsfiles-git.github.io/OLO-Aero/

---

## 📊 OLO-Dash

A daily data dashboard for your desk. Displays weather, air quality, stock prices, cryptocurrency, currency exchange rates, holidays, daylight times, and dual-city world clock. Features 15 OLO-style face animations to keep the companion personality alive.

Repository

https://github.com/micromakerlabsfiles-git/OLO-Dash

Web Dashboard

https://micromakerlabsfiles-git.github.io/OLO-Dash/

---

## 🔊 OLO-Echo

A premium OLED audio visualizer. Plug in a microphone or line-in audio source to display 17+ real-time physics-accurate visualizer themes — retro VU needles, digital bar graphs, waterfall peaks, radial wavebursts, and more. The NeoPixel LED syncs and pulses in real-time with the audio. Supports single OLED mono mode or dual OLED stereo mode.

Repository

https://github.com/micromakerlabsfiles-git/OLO-Echo

Web Controller

https://micromakerlabsfiles-git.github.io/OLO-Echo/

---

## 🖱️ OLO-Jiggler

Stealth productivity and mouse movement utility for desktop automation. Emulates a BLE mouse to keep workstations active without triggering corporate idle timeouts. Includes a Pomodoro focus timer, daily habit checklists, 100 focus quotes, and 5 jiggle movement modes with NeoPixel status sync.

Repository

https://github.com/micromakerlabsfiles-git/OLO-Jiggler

Web Controller

https://micromakerlabsfiles-git.github.io/OLO-Jiggler/

---

## ⌨️ OLO-Keystrokes

BLE keyboard emulator for IT professionals, security learners, and automation enthusiasts. Type pre-programmed scripts or payloads wirelessly to any Bluetooth device. Features an on-device payload menu, local Wi-Fi dashboard, and a stealth clock interface.

Repository

https://github.com/micromakerlabsfiles-git/OLO-Keystrokes

Web Controller

https://micromakerlabsfiles-git.github.io/OLO-Keystrokes/

---

## 🔐 OLO-Pass

An offline hardware password manager for your desk. Stores up to 100 account credentials on the device, protected by a 4-digit PIN. Uses BLE keyboard emulation to automatically type your username, Tab, password, and Enter into any paired device (PC, Mac, phone, tablet). No internet, no cloud, no app — everything is on-device. Includes a browser vault manager dashboard with CSV import/export.

Repository

https://github.com/micromakerlabsfiles-git/OLO-Pass

Web Controller

https://micromakerlabsfiles-git.github.io/OLO-Pass/

---

## 🟨 OLO-Glyph

Programmable OLED text display with 13 clock styles, IoT widget support, Wi-Fi dashboard, and animated equalizer visuals. Connect to your local network to push text banners, messages, and tones remotely via `http://ologlyph.local`. Buzzer melodies (Star Wars, Mario) sync with NeoPixel LED flashing.

Repository

https://github.com/micromakerlabsfiles-git/OLO-Glyph

Web Controller

https://micromakerlabsfiles-git.github.io/OLO-Glyph/

---

## 🎬 OLO-Show

An all-in-one 1-bit OLED animation toolchain. Convert any video or GIF into dithered 1-bit OLED frame arrays, compile them into custom firmware using the GUI builder, flash them directly from the browser via WebSerial, and control live playback and NeoPixel effects in real-time.

Repository

https://github.com/micromakerlabsfiles-git/OLO-Show

Web Controller

https://micromakerlabsfiles-git.github.io/OLO-Show/

# OLOVerse Philosophy

Build once. Connect everything.

OLOVerse is designed as a modular ecosystem where firmware, software, web tools and hardware work together through a unified user experience. Every device is developed with simplicity, performance and extensibility in mind, enabling makers, students and professionals to build smarter embedded solutions.
